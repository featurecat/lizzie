Lizzie patched-0.7.4

(English / 日本語)

(Unofficial Windows package) This is an ALL-IN-ONE package of Lizzie, one of the most popular tools for Go AI. You need no installation, no Java setup, no GPU to enjoy it.

Contents

  • Lizzie: patched-0.7.4 on 2021-07-04 (Major bugs in Lizzie 0.7.4 are fixed.)
  • KataGo: 1.9.1 + 15 blocks network (g170e-b15c192-s1672 from KataGo 1.4.5)
  • Java: minimum runtime environment for Lizzie

Usage

Right-click the downloaded file to unzip it (Never double-click the zip file directly to open it). Then just double-click "Lizzie" in it. Hit SPACE key to stop/start analysis. You may want to switch the rule from "Analyze" menu. You can also switch three engines from "Engine" menu:
  • KataGo (CPU): slow but safe for old/low-end PC
  • KataGo (Modern CPU): faster
  • KataGo (GPU): much faster if your GPU works successfully. Note that it takes long time for initial tuning at the first run.
